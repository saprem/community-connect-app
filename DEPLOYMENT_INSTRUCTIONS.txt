STEP 2: Push to GitHub
──────────────────────────────────────────────────────────────────────────
cd /Users/saprem.shah/src/github.com/saprem/community-connect-app
git push -u origin main

STEP 3: Test on Mobile (Optional but Recommended)
──────────────────────────────────────────────────────────────────────────
# Install Expo Go app on your phone from App Store/Play Store

# Start Expo development server
npx expo start

# Scan QR code with Expo Go app
# App will load on your phone instantly!

STEP 4: Build Web Version for Vercel
──────────────────────────────────────────────────────────────────────────
# Build for web
npx expo export:web

# This creates a web-build folder that can be deployed

STEP 5: Deploy to Vercel
──────────────────────────────────────────────────────────────────────────
Option A: Via Vercel CLI
  vercel --prod

Option B: Via Vercel Dashboard
  1. Go to https://vercel.com/new
  2. Import from GitHub: saprem/community-connect-app
  3. Framework Preset: Other
  4. Build Command: npx expo export:web
  5. Output Directory: web-build
  6. Deploy!

💡 PRO TIPS:

1. Test on Real Device First
   - Expo Go provides instant feedback
   - No need for Android Studio or Xcode
   - Just scan QR and see changes live!

2. Hot Reload is Automatic
   - Make code changes
   - Save file
   - App updates instantly on your phone!

3. Web Build for Demo
   - Vercel deployment good for showing stakeholders
   - But mobile experience is much better
   - Focus on mobile-first development

4. Firebase Setup When Ready
   - Free tier is generous
   - Follow SETUP_GUIDE.md for Firebase
   - No credit card needed to start

5. Incremental Development
   - Build one feature at a time
   - Test frequently
   - Get user feedback early
