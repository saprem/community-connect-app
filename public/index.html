Community Connect

Mobile App for Community Management

📱 How to Test This App

Step 1: Install Expo Go

Download "Expo Go" app from App Store (iPhone) or Play Store (Android)

Step 2: Clone Repository

git clone https://github.com/saprem/community-connect-app.git

Step 3: Install & Run

npm install && npx expo start

Step 4: Scan QR Code

Open Expo Go app and scan the QR code from your terminal

✨ Features

🚶Visitor Management - QR codes & approvals
💬Community Chat - Events & discussions
💰Payment Tracking - Maintenance & bills
📋Notice Board - Announcements & updates
🔔Smart Notifications - Real-time alerts
⚠️ This is a React Native mobile app. It requires Expo Go app on your phone to test. Web version is not available for this type of app.
